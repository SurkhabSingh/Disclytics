# Disclytics

Disclytics is a personalized Discord analytics bot and dashboard for tracking message activity, voice usage, reminders, and channel-level participation inside servers where the bot is installed.

## Live App

https://disclytics-web.vercel.app



## What Disclytics Can Do

- Personal Insights on daily activities from servers where the bot is present.
- Track messages sent in servers where the bot is installed
- Track voice sessions and time spent in voice channels
- Show today, history, and lifetime activity in the dashboard
- Show recent messages and recent voice sessions
- Show top text channels and top voice channels
- Send reminder DMs
- Provide per-server slash command stats

## Discord Commands

- `/stats`
  Shows your tracked stats for the current server
- `/remind`
  Creates a reminder that Disclytics can send back to you later

## How To Use Disclytics

1. Open the dashboard.
2. Sign in with Discord.
3. Invite the bot to the servers you want to track.
4. Start using the server normally.
5. Open the dashboard to view your activity.

## Important Rules

- Disclytics only tracks activity in servers where the bot is present.
- It does not track global Discord activity.
- It does not use self-bots or user tokens.
- It can only see channels the bot has permission to view.
